# 👋 YesHello.lol — The Gen Z Communication Power Move

A humorous website that showcases the "Yes Hello" communication style popular among Gen Z — just say "Hello" and make them wait. It's giving main character energy.

🔗 **Live site:** [yeshello.lol](https://yeshello.lol)

## ✨ Features

- **Interactive conversation examples** — see "good" vs "bad" hellos with typing animations
- **Gen Z slang glossary** — click the 👀 emoji to explore 27 terms with definitions
- **Light / Dark mode toggle** — manual ☀️/🌙 switch with `localStorage` persistence, falls back to OS preference
- **Responsive design** — optimized for mobile (380px+), tablet, and desktop
- **Accessibility** — skip-to-content link, keyboard navigation, `aria` attributes, `prefers-reduced-motion` support
- **Social sharing** — share on X/Twitter or copy link with one click
- **Term highlighting** — hover or click any Gen Z term for a visual pulse

## 🛠 Tech Stack

| Layer | Details |
|-------|---------|
| **Markup** | Semantic HTML5 with `aria` roles, lazy-loaded images |
| **Styles** | CSS3 — custom properties, `clamp()` typography, CSS-only theme switching |
| **Scripts** | Vanilla JS (no dependencies) — IIFE pattern, `localStorage` theming |
| **Hosting** | Static site — works on GitHub Pages, Netlify, Cloudflare Pages, etc. |

## 🚀 Getting Started

```bash
git clone https://github.com/Manaiakalani/yeshello.lol.git
cd yeshello.lol
open index.html   # or use any local server
```

Then:
1. Click the 👀 emoji to view the slang glossary
2. Toggle ☀️/🌙 in the top-right to switch themes
3. Explore the "good" and "bad" hello examples
